**Dotanuki Labs** is a space for high quality open-source.

Main areas of focus:

- Rust, Kotlin, Python, WebAssembly
- Developer tools, CLI apps, Web services, Bot apps
- Build Systems, CI/CD automation, Infrastructure automation
- DevSecOps and Software Supply Chains
- AI-powered tooling
- Etc

## Why Dotanuki?

[Dotanuki](https://en.wikipedia.org/wiki/D%C5%8Dtanuki) is a type of Samurai sword.
This fact was learned by many of us after reading the most awesome manga ever written :
[Lone Wolf and Cub](https://en.wikipedia.org/wiki/Lone_Wolf_and_Cub)

**Dotanuki** is a place for open-source high quality open-source, including experimental tooling,
libraries, demos and more.

Main areas of focus:

- Rust, Kotlin, Python, WebAssembly
- Developer tools, CLI apps, Web services
- Build Systems, CI/CD automation, Bot apps
- DevSecOps, Software Supply Chain tooling
- Infrastructure automation
- AI-powered tooling
- Etc

Curated by [@ubiratansoares](https://github.com/ubiratansoares)

## Why Dotanuki?

[Dotanuki](https://en.wikipedia.org/wiki/D%C5%8Dtanuki) is a type of Samurai sword.
This fact was learned by many of us after being introduced to the most awesome
manga ever written :
[Lone Wolf and Cub](https://en.wikipedia.org/wiki/Lone_Wolf_and_Cub)
